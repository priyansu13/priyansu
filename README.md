# Priyansu Kumar — Data Analyst Portfolio

## Local development

Run `npm install`, then `npm run dev` and open `http://127.0.0.1:3000`.

## Publish on GitHub Pages

1. Push this project to a GitHub repository on the `main` branch.
2. In the repository, open **Settings → Pages** and set the source to **GitHub Actions**.
3. Push to `main`. The included workflow builds the static site and publishes it automatically.

The site is configured for both a project repository (`username.github.io/repository-name`) and a user-site repository (`username.github.io`).

## Contact form

The contact form sends messages to `priyansukumar38517@gmail.com` through FormSubmit. When the first message is sent from the deployed site, FormSubmit will email Priyansu an activation link. Open that link once to activate delivery; subsequent form messages are sent to the inbox.
